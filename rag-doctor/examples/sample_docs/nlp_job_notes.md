# NLP And LLM Engineer Notes

An NLP or LLM application engineer should understand tokenization, embeddings, vector search, prompt design, retrieval augmented generation, evaluation, Python packaging, tests, and deployment.

For interviews, it is useful to explain tradeoffs. Chunk size changes retrieval behavior. Embedding model choice changes semantic recall. Reranking can improve precision. Evaluation should include retrieval quality and answer faithfulness.

中文求职准备可以围绕五个能力展开：Python 工程能力、NLP 基础、RAG 应用开发、评估方法、部署和监控。

一个好的作品集项目应该能运行、能解释、有测试、有 README、有示例数据，并且能清楚说明它解决了什么真实问题。
